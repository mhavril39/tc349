Web design principles

Rule of Thirds

Good Example: Rolling Stone's website does follow the rule of thirds, with the homepage's content divided into three manageable columns: breaking news, headline stories, and other news.

Bad Example: Sewing & Embroidery Warehouse's website does not follow the rule of thirds, with all the centered on the website. As an added bonus, this page has no h2 header tags closed, meaning the text continuously grows in size.

Chunking

Good Example: Tumblr's website has a very simple timeline, presenting you with one post at a time.

Bad Example: Arngren's website has a lot of items, but has far too much information to have any kind of useful interpretation.

Ockham's Razor

Good Example: Everlane's website is simple, clean and easy to scroll thru.

Bad Example: The Royal British Legion's website has a lot of fun and cute little animations, but overall it's just too hard to get to the actual information they're presenting.

Modularity

Good Example: Jay Z's website divides everything into nice little squares to click on.

Bad Example: Andrew Hitchcock's website is basically a big long paragraph. Yuck!

Mapping

Good Example: Think Education's website has a nice navigation at the top that stays with you as you scroll.

Bad Example: Britney Spears's website is so simple that it takes time to find the actual navigation. Everything is sleek, but where are the controls?
